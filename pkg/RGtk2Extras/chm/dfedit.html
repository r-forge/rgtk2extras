dfedit(RGtk2Extras)R Documentation

Convenience function for editing a data frame in its own window

Description

Optional name to assign the dataset when updating

Usage

dfedit(items, dataset.name = deparse(substitute(items)), 
size=c(500, 300), col.width=64, pretty_print=F)

Arguments

items The data frame to edit
dataset.name Optional name to assign the dataset when updating
size Optional window size request
col.width Optional column size
pretty_print Optional pretty_print

Details

IF set to NULL, is the name of the dataset.

Value

Returns the gtkDfEdit object

Examples

 
  obj <- dfedit(iris)
